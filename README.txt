Project Name: Inverted Pendulum Controller

Description: Firmware for the full-state feedback controller device used to automatically balance an inverted pendulum.

Target board: TM4C123GH6PM (Texas Instruments)

Toolchain: TI C Compiler

CPU: TI ARM Cortex M4
